
# EventaSports

EventaSports is a modern, high-performance tournament management platform designed to streamline the organization of esports and sports events. Built with a stunning, interactive frontend and a robust backend, it offers a seamless experience for organizers, teams, and spectators.

## Features

- **Dynamic Frontend**: Built with React and Vite, featuring smooth scrolling (Lenis) and immersive 3D backgrounds (Three.js).
- **Interactive Animations**: Powered by GSAP for a premium, responsive feel.
- **Tournament Management**: Create and manage tournaments with ease.
- **Team & Match Tracking**: Dedicated pages for viewing team details and match schedules.
- **Robust Backend**: Python Flask API handling data persistence with SQLAlchemy.
- **Modern Design**: Glassmorphism aesthetics, responsive mobile menu, and vibrant color palettes.

## Quick Start

Follow these steps to set up the project locally on your machine.

### Prerequisites

Make sure you have the following installed on your machine:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) (Node Package Manager)

### Cloning the Repository

```bash
git clone https://github.com/Chiru-Dey/binary-breakers.git
cd binary-breakers
```

### Installation

Install the project dependencies using npm:

```bash
npm install
```

### Running the Project

```bash
npm run dev
```
